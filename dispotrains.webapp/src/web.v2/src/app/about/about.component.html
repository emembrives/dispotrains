Dispotrains

DispoTrains: l'information (presque) temps-réel sur l'accessibilité du réseau ferré d'Ile-de-France aux usagers à mobilité réduite. Retrouvez l'état des ascenseurs dans les gares des réseaux ferrés d'Ile-de-France (RATP, Transilien: métro, RER et tram) et ainsi planifiez vos déplacements en évitant les surprises. Par Etienne Membrives.
Comment utiliser ce site ?

Les gares sont classées par ligne. Sélectionnez une ligne pour afficher toutes les gares accessibles. Les gares disposant d'ascenseurs hors-services sont signalées. Il est également possible d'afficher des informations détaillées sur la gare : le détail de ses ascenseurs, sa situation géographique, ainsi que les lignes disponibles.

À propos

Ce site est donné à titre indicatif uniquement. Il est possible que les informations qu'il contient soient incomplètes, inexactes ou en retard par rapport à la situation réelle. N'hésitez pas à me contacter si vous repérez de tels problèmes afin que je puisse y remédier.

Code et source de données

Le contenu de ce site est généré de manière automatique à partir d'informations fournies par Ile-de-France Mobilités. L'ensemble des données, avec l'historique complet, est disponible en téléchargement sous forme d'archive JSON (données récentes) et d'archive TSV (données historiques Infomobi). Le code source est disponible sous licence Apache 2.0 à l'adresse https://github.com/emembrives/dispotrains.
